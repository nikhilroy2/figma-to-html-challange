# Mining.com.au Design System - Getting Started

This project is a static HTML/CSS conversion of a Figma UI Kit and Design System for Mining.com.au. It serves as a "Getting Started" or "Foundations" landing page to help designers, marketers, and developers understand how the design system is structured and how to navigate the files efficiently.

## Features

- **Fully Responsive Design**: The layout adapts seamlessly across desktop, tablet, and mobile devices without losing its aesthetic appeal.
- **Modern Typography**: Uses Google Fonts (`Inter` and `Roboto`) to ensure clear and professional readability.
- **Custom Header & Footer**: Includes a full-width header with navigation and a customized responsive footer with a distinct background shape.
- **Interactive Elements**: Features clickable image covers and linking that direct to the main website.

## Technologies Used

- HTML5
- CSS3 (Vanilla)
- Google Fonts API

## How to Run

1. Clone or download this repository to your local machine.
2. Ensure all files (`index.html`, `style.css`, and the `images/` directory) are in the same folder.
3. Double-click the `index.html` file to open it in your default web browser, or use a local development server like Live Server in VS Code.

## Project Structure

- `index.html`: The main markup file containing the structure of the page.
- `style.css`: The stylesheet handling all visual styling, layout, and media queries.
- `images/`: The directory containing all the assets, logos, shapes, and cover images used in the page.
